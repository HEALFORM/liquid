# ConditionalWrapper

The `ConditionalWrapper` component allows you to wrap anything only `when` props is true. Otherwise, it will directly render the children.

Here is how it is rendered `when={1 + 1 === 2}`

<Story name="when the condition is true">
  <ConditionalWrapper when={1 + 1 === 2} wrapper={children =>
    <VStack spacing={2} p={4} bg="gray.100" id='wrapper'>
      <Text>I am the wrapper</Text>
      {children}
    </VStack>
  }>
    <Text p={2} bg="gray.50" id='children'>I am the children</Text>
  </ConditionalWrapper>
</Story>
